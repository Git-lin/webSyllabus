CSS过渡特效概述

在CSS中用于设置过渡特效的属性叫做“transition”，该属性允许CSS的属性值在一定的时间区间内平滑地过渡，这就拜托了我们在CSS3版本标准之前对“JavaScript”和“Flash”的依赖，使页面的性能得以提升。这种效果可以在鼠标悬浮（:hover）、鼠标单击（:active）、表单元素获得焦点（:focus）或对元素任何改变以及在JavaScript中某些事件执行后触发，并圆滑（若不对时间曲线进行特殊设置）地以动画效果改变CSS的属性值。

该属性能够对CSS中绝大部分属性的变化生效，但不能对“显示类型（display）”的转换生效，这点在使用中需要留意。该属性包含多个分支属性，接下来我们对它的各分支属性进行详细的学习。

执行变换的属性“transition-property”

该属性是用来指定当元素其中一个属性改变时执行的过渡动画效果，该属性有三种类型的值：

none
将过渡效果设置为“无过渡效果”，或停止当前过渡效果。
all（默认）
为所支持的所有CSS属性在值变化时执行动画过渡效果。
[property name]
指定一个或多个属性名称列表，以逗号“,”进行分隔，当指定的属性产生变化时，为其执行指定属性的动画过渡效果。

过渡时间曲线“transition-timing-function”

该属性允许你根据时间的推进去改变属性值的变换速率，目前它可能够设置6个类型的值：

ease
默认值，逐渐变慢。
linear
匀速。
ease-in
加速。
ease-out
减速。
ease-in-out
先加速，后减速。
cubic-bezier([参数])
可以定义一个时间曲线，可以为其配置四个参数，前两个参数为“x1”和“x2”，定义“开始控制点”，后两个参数为“y1”和“y2”，定义“结束控制点”。而“开始点”和“结束点”是通过这两条“转换点控制轴”分别去调整两个点来实现曲线的变化的，如果对Photoshop里面的“钢笔工具”的“路径”比较熟悉的话，稍加联想应该能理解到这个时间曲线轴的改变原理，这种曲线叫做“赛贝尔曲线”。

<章节练习>

尝试用CSS3的过渡属性制作出一个“创意特效”，具体要求如下：

1、不能是简单的色彩、大小、字体等变化，要有一定创意；

2、制作效果扩展性强，能为以后的作业或项目中所使用。
