信用卡免息制度

所谓免息制度，就是你在当月账单结算日之前产生的信用卡消费部分，在次月还款截止日期之前还款享受“0利息”的一种信用卡制度。需要特别说明的是，如果是对信用卡进行提现，那么将不能享受免息，会从你提现当日起，对提现部分的金额进行利息计算。

例如：我的每月出账日期为10号，结算日期为28号，结算日期结算的是上一个月产生的账单。所以，如果我在3月10号（如这年的3月有31天）这天出账后用信用卡产生了一笔消费，要到4月10号才出账，而这笔消费需要在4月28日前还清。也就是31（3月10日至4月10日）+19（4月10日-28日）共计50天，这50天就叫做免息期。很多银行宣传的“最多长达50天的免息期”就是这样来的。也就是说，按照这样的计算方式，一年中最少的一个月（2月只有28天的年份），也有28天的免息期（28日结算前进行了消费）。

以上例为例，当过了4月28日以后，信用卡银行系统会对没有进行分期的消费金额部分，按照欠款总额每日万分之五计算。（是每日哦，可不是每月。）

分期利率说明

概念：分期就是对单次信用卡消费的金额分为N次来偿还的一种理财方式。一个分期操作只能针对一次消费，若一个账单周期内有多次消费，则在这个账单周期内可以进行多次分期操作。如果超出当月所消费的账单结算日，则不能再对消费额进行分期了。

招商银行信用卡账单分期收费标准如下（分期数 ---- 每期利率）：

2期 1.0%

3期 0.9%

6期 0.75%

10期 0.7%

12期 0.66%

18期 0.68%

24期 0.68%

细心的用户一眼就能发现分为18期和分为24期的利率是一样的，也就是说分期数只要高于18期，那每期的利率都是一样的。不过可惜的是招行并不提供大于24期的分期数，就算你信用额度再高，在收入“稳定”的情况下，想用信用开来买车买房的话，压力还是比较大的，甚至每期还款都会超出你收入的很多。

不过需要提醒的是，虽然在一定分期数内的利息率是随着分期数的增多而减少的，但千万不要想当然的就认为分期数越多越好。这个问题其实只要稍微算一下就能清楚的看到不同分期数所需还款总额的差别，我们以10000元为例（不计本金）：
分为2期时： 总还款额 = (10000 x 1.0%) x 2 = 200 元
分为3期时： 总还款额 = (10000 x 0.9%) x 3 = 270 元
分为6期时： 总还款额 = (10000 x 0.75%) x 6 = 450 元
分为10期时： 总还款额 = (10000 x 0.7%) x 10 = 700 元
……
分为18或24期时： 总还款额 = (10000 x 0.68%) x 18 = 1124 元
总还款额 = (10000 x 0.68%) x 24 = 1632 元

通过这一系列的计算发现，同样是10000元，分为最低的2期和分为最高的24期所需还款竟有1432元的差距。所以，如果对信用卡产生了需要分期消费，那么分期数是在每月经济能承受的限度下尽量的少为好。否则，只会造成多余无效支出。当然，对于盈利的那方（信用卡所属银行）来说，自然希望你的分期数尽可能的多，从而赚取更多的利润。

需要补充的是，银行出于一些其它原因（能否盈利作者觉得是个很重要的原因，当然不只是这个原因）的考虑，对单笔消费的分期也有限制。
